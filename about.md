# PikaUtils

PikaUtils is a small Geometry Dash utility mod focused on useful extras, gameplay polish, and a little personality.

Current features:

- Dynamic Discord Rich Presence that reflects menus, gameplay, practice mode, daily levels, weekly demons, and the level editor.
- Live Discord RPC toggle through Geode settings.
- Oiia Cat death sound effect.
- Click Between Frames setting for queued player inputs between physics steps.

PikaUtils is built to stay lightweight and practical while adding features that make Geometry Dash feel a bit more personal.
